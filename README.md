## Requirements
1. php >= 5.6
2. Zoopla PEM and CERT files

## Install

To install with Composer:

```sh
composer require mehmetbulut/zoopla
```

or 

Composer json
```json
"require": {
	"composer require mehmetbulut/zoopla" : "1.*"
}
```
